The Zookeeper monitoring integration captures critical performance metrics reported by [Apache Zookeeper](https://zookeeper.apache.org/) nodes and reports data to New Relic.

To get started and for installation instructions, see the [Zookeeper On-Host Integration README](https://github.com/newrelic/nri-zookeeper) on GitHub.

<Callout variant="tip">
  This integration is released as [open source on Github](https://github.com/newrelic/nri-zookeeper). A change log is also available there for the latest updates. Support for this integration is available with an [Expert Services subscription](https://www.newrelic.com/expertservices).
</Callout>
