### Notes

Support for incident context from [New Relic Radar](https://docs.newrelic.com/docs/radar/new-relic-radar/getting-started/introduction-radar), which is a tool for faster orientation during an incident. Incident context provides faster orientation during an incident by detecting unusual behavior in your system and placing it alongside alert information.

### New features

* Radar incident context

### Improvements

* Improved Infrastructure list calls to reduce latency and network usage
