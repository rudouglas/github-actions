### Notes

Release 2.8.20 finishes up the fixes for removing the ability for SAML accounts to log in via an email token. The Android app will no longer show the "Log in without a password" button if the default account is a SAML account. This release also updates the Infrastructure product to match the data available on NewRelic website.

### New features

* Host tab for Infrastructure
* Applications per host from Infrastructure hosts table
* Violations per host from Infrastructure hosts table

### Improvements

* Improved login flow for saml accounts
