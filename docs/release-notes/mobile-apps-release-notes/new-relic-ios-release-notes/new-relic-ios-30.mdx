### Notes

New Relic for iPad!

### New features

* All-things view: see everything in one view sorted by status
* Responsive grid view: choose from 3 sizes of cell to display more or less data
* Kiosk mode: full screen view
* Favorites: save the most important applications so you can view them all in one place

### Improvements

* More charts
* Improved alerting views
