## Notes

A new version of the agent has been released. Follow standard procedures to [update your Infrastructure agent](/docs/infrastructure/new-relic-infrastructure/installation/update-infrastructure-agent).

### Added

* \[Beta] Log forwarding can now collect Syslog logs.
* \[Beta] Log forwarding can now collect logs over plain TCP socket connections.
* \[Beta] Log forwarding can now be configured via user-provided Fluent-Bit configuration files.
* \[Beta] Log forwarding is now able to include custom attributes for each log record as key-value pairs.

For more information, see [Forward your logs using New Relic Infrastructure](https://docs.newrelic.com/docs/logs/enable-new-relic-logs/1-enable-logs/forward-your-logs-using-new-relic-infrastructure).
