## Notes

A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).  
New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months.

### Fixed

* [#275](https://github.com/newrelic/infrastructure-agent/pull/275) SLES-11 service management clean stops/starts.
* [#249](https://github.com/newrelic/infrastructure-agent/pull/249) Agent configuration plugin serialises only fields defined as public (struct annotation).

### Changed

* [#285](https://github.com/newrelic/infrastructure-agent/pull/285) nri-flex updated to [v1.3.9](https://github.com/newrelic/nri-flex/releases/tag/v1.3.9).

### Added

* [#286](https://github.com/newrelic/infrastructure-agent/pull/286) [Fedramp configuration flag](https://docs.newrelic.com/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings#fedramp) (`fedramp: true`) & remove gov license prefix.
* [#287](https://github.com/newrelic/infrastructure-agent/pull/287) Command channel `run/stop_integration` allow list.
