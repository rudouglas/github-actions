## Notes

A new version of the agent has been released. Follow standard procedures to [update your infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

### Added

* [#131](https://github.com/newrelic/infrastructure-agent/pull/131) Dimensional metrics will be sent using `collector_url` configuration for fedRAMP compliance.

### Fixed

* [#137](https://github.com/newrelic/infrastructure-agent/issues/137) A bug in how `collector_url` is calculated to be FedRAMP compliance.

For more information FedRAMP compliance, [see our docs](https://docs.newrelic.com/docs/security/security-privacy/compliance/fedramp-compliant-endpoints#infrastructure-endpoints).
