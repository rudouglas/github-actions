### New

* The following integrations have been released as beta:
  * \[Beta] [AWS Amazon MQ](/docs/integrations/amazon-integrations/aws-integrations-list/aws-mq-integration)
  * \[Beta] [AWS MSK](/docs/integrations/amazon-integrations/aws-integrations-list/aws-managed-kafka-msk-integration)
  * \[Beta] [AWS Amazon Athena](/docs/integrations/amazon-integrations/aws-integrations-list/aws-athena-monitoring-integration)
  * \[Beta] [AWS Step Functions](/docs/integrations/amazon-integrations/aws-integrations-list/aws-step-functions-monitoring-integration)
  * \[Beta] [AWS Direct Connect](/docs/integrations/amazon-integrations/aws-integrations-list/aws-direct-connect-monitoring-integration)
  * \[Beta] [AWS Amazon DocumentDB](/docs/integrations/amazon-integrations/aws-integrations-list/aws-documentdb-monitoring-integration)
  * \[Beta] [AWS WAF](/docs/integrations/amazon-integrations/aws-integrations-list/aws-waf-monitoring-integration)
  * \[Beta] [AWS AppSync](/docs/integrations/amazon-integrations/aws-integrations-list/aws-appsync-monitoring-integration)
  * \[Beta] [AWS QLDB](/docs/integrations/amazon-integrations/aws-integrations-list/aws-qldb-monitoring-integration)
  * \[Beta] [AWS GovCloud](/docs/integrations/amazon-integrations/get-started/connect-aws-govcloud-new-relic)
* Direct authentication with API keys is now available for AWS integrations
* Account status dashboard now features the following data:
  * Authentication errors
  * Permission errors
