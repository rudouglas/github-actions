### New In This Release

* Starting with this version of the SDK, you can set a custom user identifier value that can be used to associate user sessions with analytics events and attributes. Check out the [NewRelic.setUserId](https://docs.newrelic.com/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api#setUserId) method for more information.
