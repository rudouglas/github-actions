### Notes

* Added support for slow query traces. Slow datastore segments will now generate slow query traces viewable on the datastore tab. These traces include a stack trace and help you to [debug slow datastore activity](https://docs.newrelic.com/docs/apm/applications-menu/features/analyze-database-instance-level-performance-issues).
* Added new [DatastoreSegment](https://godoc.org/github.com/newrelic/go-agent#DatastoreSegment) fields `ParameterizedQuery`, `QueryParameters`, `Host`, `PortPathOrID`, and `DatabaseName`. These fields will be shown in transaction traces and in slow query traces.
