## 2.10.0

### New Features

* Added support for custom events when using [nrlambda](https://godoc.org/github.com/newrelic/go-agent/_integrations/nrlambda). Example Lambda handler which creates custom event:

  ```
  func handler(ctx context.Context) {
      if txn := newrelic.FromContext(ctx); nil != txn {
          txn.Application().RecordCustomEvent("myEvent", map[string]interface{}{
              "zip": "zap",
          })
      }
      fmt.Println("hello world!")
  }
  ```
