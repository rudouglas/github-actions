The PHP agent API allows you to extend the functionality of the PHP agent. The agent API is included by default with your installation, so no additional configuration is required to use the agent.

Among other things, you can use the API to [customize your app name](/docs/agents/php-agent/php-agent-api/newrelic_set_appname); [collect errors](/docs/agents/php-agent/php-agent-api/newrelic_notice_error); record [custom attributes](/docs/agents/php-agent/php-agent-api/newrelic_add_custom_parameter), [custom events](/docs/agents/php-agent/php-agent-api/newrelic_record_custom_event), and [custom metrics](/docs/agents/php-agent/php-agent-api/newrelic_custom_metric); and [enable](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header) or [disable](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum)Â browser monitoring. For an introduction to this API, see [Guide to PHP agent API](/docs/agents/php-agent/api-guides/guide-using-php-agent-api).

<Callout variant="tip">

Enclose these API calls in a conditional check for the PHP agent so that your code will work even if the agent is not present:

```php
if (extension_loaded('newrelic')) {
  newrelic_set_appname($name);
}
```

</Callout>
