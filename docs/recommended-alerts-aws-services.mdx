New Relic Infrastructure provides a collection of predefined alerts to help you monitor data from supported AWS integrations. Receive notifications for errors or fluctuations in performance, and refer to the runbook for each alert for recommended actions. To see the features of specific integrations and the data you can collect, see the [AWS integrations list](/docs/infrastructure/amazon-integrations/aws-integrations-list).
